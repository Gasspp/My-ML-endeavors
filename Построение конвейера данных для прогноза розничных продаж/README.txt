########### Данный проект представляет собой пайплайн для воспроизвдения полного     ########### 
########### цикла создания данных, обучения модели и последующей визуализации данных ########### 

!!!ВАЖНО!! Все файлы должны располагаться в одной папке 

Список необходимых пакетов и зависимостей для корректной работы содержимого:
├── numpy
├── pandas 
├── matplotlib
├── seaborn
├── plotly
├── sklearn
├── catboost
├── lightgbm
├── prophet
├── streamlit

Структура проекта:

├── .ipynb_checkpoints        			# Системный файл, хранящий метаданные работы JupyterNotebook
├── catboost_info             			# Папка с подробным отчетом обучения предобученной модели CatBoost
├── catboost_model.cbm        			# Предобученная модель CatBoost
├── catboost_model.json       			# json версия сохраненной модели
├── data-generation-for-predicitons-script.py	# Скрипт, генерирующий данные для предсказаний
├── data-generation-for-train-script.py		# Скрипт, генерирующий данные для обучения данных
├── EDA и обучение модели.ipynb			# Ноутбук с шагами обучения модели
├── sales_data_for_prediction.csv		# Готовый файл, который можно сразу же использовать для предсказаний
├── sales_data_for_train.csv			# Данные, которые использовались для обучения предобученной модели
├── streamlit_vis.py          			# Скрипт, реализующий визуализацию данных
├── README.md                 			# Описание проекта


## ШАГИ ВЫПОЛНЕНИЯ ПРОЕКТА ##

## ШАГ 0 ##
Установите необходимые зависимости. Откройте командную строку (cmd) и введите:

>>> pip install numpy pandas matplotlib seaborn plotly sklearn catboost lightgbm prophet streamlit

Или же установите каждую из библиотек по отдельности
## ШАГ 1 ##

Запустите скрипт 
├── data-generation-for-train-script.py	
Со следующими вводными данными: 

Введите сид для воспроизводимости данных: 42
Введите длину данных: 8760 
Введите количество уникальных значений (до 500), не превышающее длину данных: 400 

ИЛИ 

Используйте сразу же готовый файл
├── sales_data_for_train.csv

## ШАГ 2 ##

Откройте 
├── EDA и обучение модели.ipynb
И выполните все шаги, описанные в файле

ИЛИ 

Используйте готовую предобученную модель
├── catboost_model.cbm 


## ШАГ 3 ##
Откройте консоль и введите:
>>> streamlit run "Путь\к файлу\streamlit_vis.py"

У вас должно открыться окно в браузере.

## ШАГ 4 ##
Перетащите в открытое окно уже сгенерированый
├── sales_data_for_prediction.csv

ИЛИ

Создайте свой, используя скрипт:
├── data-generation-for-predicitons-script.py

!!! ВАЖНО !!! Четко следуете инструкциям при вводе данных.

## ШАГ 5 ## 

Теперь у Вас открывается окно с визулизацией прогнозов и с возможностью скачать файл!
